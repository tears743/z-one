## 字段定义（必填）

| 字段名 | 英文字段 | 类型 | 说明 |
|---|---|---|---|
| 平台 | platform | string | 数据来源平台名称（如 微博/抖音/B站/知乎/X/Reddit 等） |
| 排名 | rank | integer | 平台内排名（1..N）。若为聚合榜，可用综合排名并注明口径 |
| 话题 | topic | string | 热点话题标题/关键词/标签 |

| 链接 | url | string | 话题页/榜单条目链接/主要信息源链接（优先可复现的公开URL） |

| 地区 | region | string | 地区/市场（如 CN、US、Global；或更具体如 CN-Guangdong）。需与平台与时区逻辑一致 |
