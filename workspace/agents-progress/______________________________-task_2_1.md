## 黄金的核心宏观驱动摘要（传导链条）
- **实际利率（10Y TIPS real yield）**：黄金的“贴现因子/机会成本”。一般而言**实际利率上行→黄金承压**；实际利率下行（降息预期、增长走弱、避险买盘）→黄金受益。  
- **美元（DXY）**：黄金以美元计价的“外汇因子”。通常**DXY走强→压制黄金**，DXY走弱→抬升黄金；但在极端风险事件中，黄金与美元可能**同涨**（避险分流/流动性结构决定幅度）。  
- **通胀预期（10Y BEI / 5Y5Y）**：影响“名义利率拆分”与黄金的抗通胀需求。**通胀预期上行且实际利率不跟随上行**（即 real rate下行/不变）时，对黄金最友好。  
- **增长预期**：通过美联储反应函数影响前端利率与实际利率；增长走弱通常带来**降息定价→实际利率下行→利多黄金**。  
- **风险偏好/尾部风险**：通过避险需求与风险溢价直接影响黄金（尤其短期）；同时也会改变美元与实际利率的相关性结构。

## 按时间维度：主导因子与可能路径

### 1) 3个月（交易与再定价窗口）
**主导因子：风险偏好/事件冲击 + 前端政策预期对实际利率的边际变化（实质上仍落到“实际利率 + DXY”）**  
- **路径A（利多黄金）**：数据转弱/金融条件收紧/地缘升级 → 市场快速计入更早降息 → **10Y实际利率下行**；风险溢价上升带来避险买盘；DXY可能先强后弱（或同强）。  
- **路径B（利空黄金）**：通胀数据再走强或美联储鹰派沟通 → 降息预期后移 → **实际利率上行**；利差支撑DXY走强 → 黄金承压。  
- **路径C（“美元与黄金同涨”尾部）**：系统性风险/流动性挤兑 → DXY走强但避险买黄金也强；最终表现取决于**真实利率是否下行**以及流动性压力大小。

### 2) 1年（政策周期主段）
**主导因子：实际利率趋势（由降息节奏+增长/通胀组合决定）与DXY的中期方向**  
- **路径A（软着陆+温和降息）**：通胀回落但不失速 → 名义利率缓降、BEI相对稳 → **实际利率温和下行/持平**；DXY从高位震荡走弱的概率上升 → 黄金偏强、以“回撤买入”型行情为主。  
- **路径B（增长明显下滑/衰退定价）**：降息更快、更深 → **实际利率显著下行**；风险偏好下降提升避险需求；DXY短期偏强但在降息兑现后可能转弱 → 黄金趋势性走强概率最高。  
- **路径C（再通胀/增长韧性）**：通胀黏性或能源推升 → 降息推迟、期限溢价抬升 → **实际利率上行或维持高位**；DXY偏强 → 黄金相对承压或高位震荡下行。

### 3) 3年（新均衡/结构因子主导）
**主导因子：长期实际利率中枢（r* + 期限溢价）与美元结构性强弱；通胀预期“锚是否松动”决定黄金是否获得长期风险溢价**  
- **路径A（实际利率中枢下移）**：生产率不及预期/财政金融压制导致实际利率回落 → **10Y实际利率中枢下降**；若通胀预期保持锚定或略高 → 黄金具备更高的估值中枢。  
- **路径B（“高r* + 高期限溢价”）**：财政供给抬升期限溢价、增长与生产率强 → **实际利率中枢抬升**；美元相对强势 → 黄金的长期回报更多依赖尾部风险事件而非利率趋势。  
- **路径C（通胀锚松动/制度风险溢价上升）**：5Y5Y上行或通胀风险被系统性重估，同时政策难以把实际利率压得足够高 → 黄金获得“抗通胀+避险”的**结构性溢价**（即便名义利率不低）。

### 一句话落地（便于团队统一口径）
- **3个月看“事件与再定价”：风险偏好→前端预期→10Y实际利率与DXY的边际变化。**  
- **1年看“政策与周期”：降息节奏决定实际利率趋势，DXY跟随利差与增长相对优势。**  
- **3年看“新均衡”：r*/期限溢价与通胀锚的稳定性决定黄金长期估值中枢。**
