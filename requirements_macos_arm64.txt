# 性能注意事项
###############################################################################

# 1. MPS 后端目前不支持所有 PyTorch 操作
#    某些操作可能会自动回退到 CPU
#
# 2. 内存管理:
#    - Apple Silicon 使用统一内存架构
#    - 设置环境变量: export PYTORCH_MPS_HIGH_WATERMARK_RATIO=0.8
#
# 3. 预期性能:
#    - 相比 NVIDIA RTX 4090: 慢 2.5-3.5x
#    - 但完全可用于开发和原型设计
#
# 4. 已知 MPS 限制:
#    - 某些 float64 操作不支持
#    - 某些复杂索引操作可能失败
#    - 如果遇到问题，可设置: export PYTORCH_ENABLE_MPS_FALLBACK=1
